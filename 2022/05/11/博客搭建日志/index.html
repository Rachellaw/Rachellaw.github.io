博客搭建日志

一、Hexo+github博客搭建

1.Node.js、git和Hexo下载

Node.js官网下载

git镜像下载

git配置

Hexo安装与测试

2.搭建仓库

create a new repository in github，repository name为username.github.io，勾选add a readme file

3.生成SSH keys

在任意一个文件夹内右键打开Git Bash Here

在C盘用户文件夹中找到.ssh文件夹，打开后找到id_rsa.pub，用记事本打开并复制内容。将内容粘贴到github里设置中的SSH keys。

4.本地生成博客内容

建立Blog文件夹存放博客本地文件，用Git Bash Here部署

5.上传到github

修改配置文件_config.yml中deploy部分

完成！即可用username.github.io访问博客。
